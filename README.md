# Exercícios de Lógica em Python

Repositório com exercícios desenvolvidos durante meus estudos em Ciência da Computação.

## Conteúdos praticados
- Entrada e saída de dados
- Conversão de tipos
- Estruturas condicionais
- Operações matemáticas

- Par e ímpar
- Múltiplos
- Validação simples de entrada

## Objetivo
Fortalecer a base em lógica de programação e desenvolver raciocínio para problemas mais estruturados.
